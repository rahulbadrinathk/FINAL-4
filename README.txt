GST Bill Tracker - GitHub Pages Ready

This package is already converted into a static website.
You do NOT need npm, Vite, React, or build commands.

How to use:
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, open Settings > Pages.
4. Under Build and deployment:
   - Source = Deploy from a branch
   - Branch = main
   - Folder = /(root)
5. Save.
6. Wait a minute, then open your website link.

Your data is stored in the browser using localStorage.
That means bills stay saved in the same browser unless you clear browser data.
